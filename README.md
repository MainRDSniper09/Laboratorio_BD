# Laboratorio_DB

Proyecto plano de python en donde se usa POO y DB con postgresql

El proyecto consiste en crear un objeto persona el cual permita realizar algunos comandos sql
entre estos esta:

1.INSERTAR
2.SELECCIONAR
3.MODIFICAR
4.ELIMINAR

## Caracteristicas
- uso de CRUD
- libreria psycopg2
- uso de POO
